Anatomy of a CSS Rule

Subheading 1

CSS works by associating rules with HTML elements. These rules govern how the content of specified elements should be displayed. A CSS rule contains two parts: a selector and a declaration.


Subheading 2

Types of selectors -

  • Element Selector (format --> tag_name_only { rules })
    • When all the elements like <p>, <div>, etc. have been defined with a style.


  • Class Selector (format --> .class_name { rules })
    • When any element uses/implements the class rules during the opening tag.

  • id Selector (format --> #id_name { rules })
    • When rules are defined for any particular id. One id can't be repeared in a single HTML file.

  • Group Selectors (format --> .class_name, .class_name, #id_name, tag_name_only { rules })
    • When multiple selectors have the same block of rules implemented. This can be done with a comma.
